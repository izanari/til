# npm
## 使い方
- バージョンを表示する
  ```
  npm --version
  ```
- パッケージをインストールする
  ```
  npm i パッケージ名
  npm i -g パッケージ名
  ```
  - `-g`をつけるとグローバルにインストールする
- パッケージをアンインストールする
    ```
    npm un パッケージ名
    ```
- パッケージを表示する
  ```
  npm ls
  npm ls -g
  ```
- アップデートする
  ```
  npm up パッケージ名
  ```
- 最新化する
  ```
  npm --version
  npm install -g npm
  npm rebuild
  ```
